# USCIS Immigration Analytics

## Project Overview
This project explores immigration case processing data to identify workload patterns and opportunities for predictive case complexity scoring.

## Business Objective
Improve operational efficiency by applying analytics to case volume and processing time data.

## Techniques Used
- Data Cleaning & Transformation
- Feature Engineering
- Trend Analysis
- Predictive Modeling Framework Design

## Strategic Value
Supports data-driven decision making in public-sector immigration services.

## Tools
Python, SQL, Tableau, Databricks

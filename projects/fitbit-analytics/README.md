# Fitbit Performance Analytics

## Project Overview
This project analyzes wearable fitness tracking data to uncover behavioral trends and build predictive models for calorie expenditure.

## Business Objective
Transform raw activity metrics into actionable performance intelligence using regression modeling and exploratory data analysis.

## Techniques Used
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Multiple Linear Regression
- Time-Series Trend Analysis

## Key Results
- Strong correlation between active minutes and calorie burn

- Behavioral rhythm patterns identified
